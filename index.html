3DS-Optimized Pokemon Type Chart

As both a project to learn about web design from a wholly different kind of device and as a tool to use with the rerelease of the Game Boy Pokemon games on the 3DS eShop, the 3DS-Optimized Pokemon Type Chart can be accessed with the 3DS Browser to determine type matchups and effectiveness.

The simple interface provides enough information to understand which moves to use against foes while respecting the 3DS's limited hardware.

For anyone interested in web development for the 3DS, a useful resource is Nintendo's own 3DS Browser specification page.
